# KCursors

## Qu'est-ce que c'est ?

**KCursors** est un pack de curseurs inspiré de la **Karmine Corp** et agrémente votre expérience PC avec la plus belle des couleurs : **LE BLEU**
  
## Comment installer le pack de curseurs KCursors sous Windows ?

### Étape 1 : Téléchargez le pack de curseurs

1. Cliquez sur le bouton **Code** en haut de cette page GitHub et sélectionnez **Download ZIP**.
2. Extrayez le contenu du fichier `.zip` sur votre ordinateur dans un dossier que vous trouverez facilement.

### Étape 2 : Installer le pack de curseurs

Une installation automatique est disponible via un fichier `.inf`, suivez les étapes ci-dessous :
1. Dans le dossier téléchargé, trouvez le fichier **Install.inf**.
2. Faites un clic droit dessus et sélectionnez **Installer**.
3. Ouvrez ensuite le Panneau de configuration, allez dans les paramètres de souris, et sélectionnez le schéma de curseur "KCursors" que vous avez installé.

Et voilà, vous avez maintenant un pack de curseurs unique aux couleurs de la Karmine Corp !


#### Si l'installation automatique ne fonctionne pas vous pouvez le faire manuellement

1. **Ouvrez le Panneau de configuration** :  
   - Appuyez sur `Win` + `R` pour ouvrir "Exécuter".
   - Tapez `control` et appuyez sur **Entrée**.
  
2. **Accédez aux paramètres de la souris** :  
   - Cliquez sur **Matériel et audio** puis sur **Souris** (dans la section "Périphériques et imprimantes").

3. **Changer les curseurs** :
   - Dans l'onglet **Pointeurs**, sélectionnez chaque type de curseur que vous souhaitez changer (par exemple, "Sélection normale", "Travail en arrière-plan", etc.).
   - Cliquez sur **Parcourir** et accédez au dossier où vous avez extrait les curseurs.
   - Choisissez les fichiers correspondants :
     - **kc_pointer.cur** pour la sélection normale,
     - **kc_work.ani** pour le travail en arrière-plan,
     - **kc_busy.ani** pour la sélection occupée, etc.

4. **Appliquez votre nouveau thème de curseur** :
   - Après avoir sélectionné chaque curseur, cliquez sur **Enregistrer sous...** et donnez un nom à votre nouveau schéma de curseur (par exemple, "KCursors").
   - Cliquez sur **OK** pour appliquer.
